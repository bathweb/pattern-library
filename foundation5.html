Lists

We use list elements in a number of ways on the University of Bath website.

There are 3 main types of list:

Unordered lists

In an unordered list each item is preceeded by a bullet, disc or other symbol.

  • The first list item
  • The first list item
    • Colons
    • Dots (ellipses)
    • Hyphens
    • Quotation marks
    • Slashes
  • The third list item
HTML example
        <ul>
            <li>The first list item</li>
            <li>The second list item</li>
            <li>The third list item</li>
        </ul>

Ordered lists

In an ordered list each item is preceeded by a number.

  1. The first list item
  2. The second list item
  3. The third list item
HTML example
   <ol>
          <li>The first list item</li>
          <li>The second list item</li>
          <li>The third list item</li>
     </ol>
